# EchoLocate Design Notes

EchoLocate is a moon-city cyber forensics game focused on spatial investigation.

## Core Pillars

- The moon-city map is the main board
- Buildings correspond to real systems and incident locations
- Specialist agents help the player analyze evidence
- The case board is used to connect findings into a coherent explanation
- Recruiting and role unlocks expand the available investigation toolkit

## Core Roles

- `LOGIS` — log analysis
- `NEXUS` — network tracing
- `FILER` — artifact and file recovery
- `CHRONO` — timeline reconstruction

## Active UX Goals

- clean landing flow into investigate mode
- readable map-first interface
- spatially accurate building selection and highlighting
- consistent agent ownership / unlock / deployment model
- believable lunar infrastructure theme
